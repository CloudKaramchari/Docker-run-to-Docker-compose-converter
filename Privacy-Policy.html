Privacy Policy

Docker Run to Docker Compose Converter

Effective Date: June 2025

Data Collection

This extension does not collect any personal data, browsing activity, Docker commands, or any other user information.

How the Extension Works

All operations, including the Docker run command conversion and YAML generation, are performed locally within your browser. No data is sent to any external servers or third-party services.

Third-Party Services

This extension does not use any third-party APIs, analytics tools, or external integrations.

Changes to This Policy

If there are any future updates to this privacy policy, they will be posted on this page.

Contact

If you have any questions about this privacy policy, please contact the developer directly via the Chrome Web Store support page.
